# Chat UI

이 프로젝트는 **Spring AI 활용 실습을 위한 웹 프론트엔드**입니다. 

## 📌 설치 및 실행

### 1️⃣ 리포지토리 클론 및 이동
```bash
git clone https://github.com/revfactory/chat-ui.git
cd chat-ui
```

### 2️⃣ 의존성 설치
```bash
npm install
```

### 3️⃣ 애플리케이션 실행
```bash
npm run dev
```
브라우저에서 `http://localhost:3000`을 열어 애플리케이션을 사용할 수 있습니다.
